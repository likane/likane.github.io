Solar System
A single page application showing the orbits of Mercury, Venus, Earth and the Earth's moon rotating around the sun. Technology used is HTML5 Canvas, css and javascript.

Forex
A single page application using an API to get currency exchange data and display foreign currency values relative to the US Dollar. The next step will be to get a working currency calculator.

Caesar Cypher
a command line node application prompting a user to provide a string and key position. The initial string is then encrypted based on the key position chosen. Caesar Cypher is one of the oldest encryption techniques developed.

Spot Finder
A RESTful, full stack application using Sequelize and Bootstrap. Spot Finder is a one stop site to try to reconnect pet owners with their pets. One of four developers on the team.

Guest List
Currently in development, Guest List allows venues to modernize and automize the guest list experience for attendees and box office personnel. Using React.js/React-router and Materialize for the front end and Mongoose ORM for the back end, this application allows users to sign in and interact intuitively and efficiently.
